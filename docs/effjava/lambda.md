# Labmda Expression


람다 수식에 대해서 살펴보자.

단 하나의 메소드만 포함하는 인터페이스와 같이 익명 클래스(anonymous class)가 매우 심플하면, 그 익명 클래스는 명확하지 않고 다루기가 까다롭다.

이런 경우, 다른 매소드의 인자로 함수(기능)를 전달하는 방법을 시도할 수 있다. 람다 수식들은 함수를 메소드 인자로, 또는 코드를 데이타로 처리할 수 있게 한다.
