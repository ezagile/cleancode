# Welcome to MkDocs

참고: [mkdocs.org](https://www.mkdocs.org).

## 명령어

* `mkdocs new [dir-name]` - 새 프로젝트 생성
* `mkdocs serve` - live-reloading 서버 시작
* `mkdocs build` - 문서 생성
* `mkdocs -h` - 도움말 출력
* `mkdocs gh-deploy
